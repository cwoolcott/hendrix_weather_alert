2) Make it executable (required)
chmod +x /home/pi/start-weather-server.sh

3) Tell the desktop to run it on boot

Create an autostart entry:

mkdir -p /home/pi/.config/autostart
nano /home/pi/.config/autostart/weather-server.desktop


Paste:

[Desktop Entry]
Type=Application
Name=Weather Server
Exec=/home/pi/start-weather-server.sh
X-GNOME-Autostart-enabled=true


Save and exit.

4) Reboot and verify
sudo reboot
